# Customer Churn Prediction 

A data analysis + machine learning project to predict 
which bank customers are likely to churn.

## Project Overview
- **Dataset:** 10,000 bank customer records
- **Goal:** Predict customer churn using AI/ML models
- **Best Model:** Random Forest — 86.65% accuracy

## Key Findings
- Overall churn rate: 20.4%
- Germany has highest churn (~32%)
- Inactive members churn 2x more than active members
- Customers aged 51–60 are highest risk segment

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn (Logistic Regression, Random Forest)
- Matplotlib, Seaborn
- Google Colab

## Project Structure
- `customer_churn_prediction.ipynb` — Main notebook
- `Churn_Modelling.csv` — Dataset
